# Guide interactif Yassmine

Ce dossier contient un guide HTML autonome pour aider Yassmine à lancer, utiliser et dépanner Mon PF App.

## Fichier principal

Ouvrir dans un navigateur :

```text
docs/yassmine-guide/index.html
```

## Contenu

- explication simple de l'architecture Flutter + Laravel ;
- lancement backend Laravel ;
- lancement Flutter sur Android réel, émulateur, Windows et Web ;
- comptes de démonstration ;
- scénario d'utilisation client, admin, serveur et livreur ;
- assistant debug Gemini avec saisie de clé API côté navigateur ;
- réponses locales sans clé Gemini pour les erreurs fréquentes.

La clé Gemini n'est jamais stockée dans le dépôt.
